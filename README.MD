# Content for GitHub Pages site

Site: https://pilat.github.io/

Howto:

```
brew install hugo
hugo new site .
git clone https://github.com/panr/hugo-theme-terminal.git themes/terminal\n
```

New page:
```
hugo new posts/hello_world.md
```

Serve locally:
```
hugo server -D
```

Before publish:
```
hugo
```
